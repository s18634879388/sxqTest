a 可点击

a 用户点击“只看楼主”按钮，动作选单自上向下推出。帖子详情页置灰取消。帖子详情页只显示楼主相关。“只看楼主”状态变为“查看全部”

c 默认状态为查看全部，按钮默认状态为“只看楼主”。
